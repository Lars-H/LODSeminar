Request

Desciption:

The compare method can be used in order to compare a given value and unit to another resource.

Accepts:

Acceptable Datatypes: "application/ld+json"

Parameters:

  • value (required)

    • Type: float

    Example: 12993

  • unit (required)

    • Type: enum

    Unit: [t,k,g,km,m,euro,usd]

    Example: "t"

HTTP Status code 200

HTTP Status code 400

HTTP Status code 406

Client Error: The accept header was not correctly specified. Please note that only "application/ld+json" is supported with the current API.

Server Error: Something went wrong on the server. We are sorry that we could not complete your request!

Example

The following example request a value of "2000000" and a unit of "t" (tons):

 http://cepler.org/compare?v=2000000&u=t
