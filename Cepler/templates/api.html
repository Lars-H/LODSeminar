Cepler API version 1.0

http://cepler.org/

/compare
  • Request
  • Response
  • Example
  • GET

Request

Desciption:

The compare method can be used in order to compare a given value and unit to another resource.

Parameters:

  • value (required)

    • Type: float

    Example: 12993

  • unit (required)

    • Type: enum

    Unit: [t,k,g,km,m,euro,usd]

    Example: "t"

Response

HTTP Status code 200
